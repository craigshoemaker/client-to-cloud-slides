Client to the Cloud

Synchronizing the in-browser database with the cloud

Working with in-browser databases gives you the opportunity to save large amounts of data without ever needing to contact the server, but for many applications the data can't just stay on the client. To bridge the gap between local and server data you need a reliable strategy that synchronizes the client to the server.

Learn to use PouchDB with CouchDB to synchronize data, deal with conflicts and untether yourself from the network.
